Assumed software present on CSIL Computers:
===========================================
- Python 2.7 (Filepath: "C:\Python27\python")
- Java JRE/JDK 1.7
- Windows Vista or newer
- Microsoft SQL Server 2003 or newer
- Eclipse IDE


Connections to External Servers:
================================
- SFU Cypress Database


Instructions to Build from Included Files:
==========================================
- Unzip the project archive.
- Open Eclipse and create a project based in the top level of the decompressed folder.
- Appropriate JAR libraries will be included automatically by Eclipse.
- Export the project as a runnable JAR file. (Make sure to click 'package required libraries...' option)


Last Two Parts of Getting Started Section:
==========================================

Running the Application:
------------------------
- Double-click on the provided "MarkShark.jar"
- Confirm network connection.
- Log into the system with a valid user account.
- Follow the instructions provided in this manual for the desired use case.

Application Installation for UAT:
---------------------------------
- Unzip the project archive.
- Open Eclipse and create a project based in the top level of the decompressed folder.
- Appropriate JAR libraries will be included automatically by Eclipse.
- Export the project as a runnable JAR file. (Make sure to click 'package required libraries...' option)
- If desired, run the MarkShark.jar file to start the program.
